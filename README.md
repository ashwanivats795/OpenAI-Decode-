📌 Project Title: OpenAI Instagram Data Analyze using Python

📖 Description
This project is developed using Python to extract and process Instagram profile data of "OpenAI" such as the number of posts, followers, and following. The program reads raw data (scraped or collected), parses it efficiently, and converts it into structured format for analysis. It also handles different formats of numerical values like thousands (K) and millions (M), ensuring accurate data processing.

🚀 Features
Extracts Instagram user data (posts, followers, following)
Cleans and processes raw text data
Handles numeric formats like K (thousands) and M (millions)
Stores data in structured format (dictionary/list)
Identifies users with maximum posts or followers

🛠️ Technologies Used
Python
String Manipulation
Data Processing Techniques

📂 How It Works
Input raw Instagram data (chunks format)
Parse each chunk using a custom Python function
Clean and convert string data into numeric values
Store processed data into a list
Perform analysis (e.g., highest followers, posts)

💡 Learning Outcomes
Improved Python programming skills
Understanding of data parsing and cleaning
Handling real-world inconsistent data formats
Working with lists, loops, and functions

🧠 Additional Functionality

This project not only extracts Instagram data but also performs basic analytical operations such as identifying users with maximum posts, followers, and following, making the dataset more meaningful and useful.

📎 Conclusion
This project demonstrates how Python can be used for real-world data extraction and preprocessing tasks, especially for social media analytics.
